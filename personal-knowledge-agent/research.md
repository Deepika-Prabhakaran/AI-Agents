# AI Research Notes

## Reinforcement Learning from Human Feedback (RLHF)

RLHF is a technique used to align language models with human preferences. It involves training a reward model on human comparisons, then using RL to optimize the LLM against that reward signal.

## Chain of Thought Prompting

Chain of Thought (CoT) prompting encourages models to reason step by step before giving a final answer. It significantly improves performance on multi-step math and logic tasks.

## Agents and Tool Use

Modern AI agents can call external tools like search engines, calculators, and code interpreters. This extends their capabilities far beyond what is possible with language alone.

## Embeddings

Word and sentence embeddings map text to dense vectors in a high-dimensional space. Semantically similar text tends to cluster together, enabling semantic search and retrieval.

## Context Window

The context window defines how many tokens a model can process in a single pass. Larger context windows allow models to reason over longer documents and conversations.
